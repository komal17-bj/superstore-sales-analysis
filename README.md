# 🛒 Superstore Sales Analysis

## 📌 Project Overview
End-to-end analysis of 10,000+ orders from a US retail company 
using Python, Pandas, and Matplotlib to identify key business insights.

## 🔧 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- GitHub

## ❓ Business Questions Answered
1. Which region generates the most revenue but lowest profit margin?
2. Which sub-categories are losing money?
3. Does higher discount lead to more loss?
4. Which customer segment is most valuable?
5. Which category ships fastest?

## 📊 Key Findings
- **West region** leads in revenue ($725K) but **Central** has lowest profit margin (7.9%)
- **Tables, Bookcases & Supplies** are loss-making sub-categories
- Discounts **above 20%** consistently result in negative profit
- **Consumer segment** drives 55% of total sales
- All categories ship in approximately **4 days**
